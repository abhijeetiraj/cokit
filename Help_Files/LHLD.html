LHLD addr:
          
          LHLD is an abbrevation for load HL directly.This instruction copies the contents of memory pointed by
          specified address into register L.The address is then incremented and the contents of this memory is
          copied into register H.
             
                For e.g.   LHLD 2000H          ;  copies the contents of memory location 2000H into L and the                                                                ;   contents of of memory location 2001H into H.
