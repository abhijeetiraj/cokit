MOV :
         
    This is an abbreviation for MOVE. It can be further subdevided into three categories.
     a) MOV r1,r2
     b) MOV M,r
     c) MOV r,M      
   
     a) MOV r1,r2  (destination,source)
          This instruction moves data from one register to another.
            For e.g.  MOV A,B        ;  Move contents of register B to register A or accumulator.
     
     b) MOV M,r
           This instruction moves data from any 8 bit register specified to memory pointed by HL register pair.
             For e.g  MOV M,A       ;  Move contents of accumulator to memory pointed by HL.
   
     c) MOV r,M
           This instruction moves data from memory pointed by HL register pair to the specified 8 bit register.
             For e.g  MOV M,A       ;  Move contents of memory pointed by HL register pair to register B
