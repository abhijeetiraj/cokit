Summary of 8085 Instruction Set
The Instruction Set can be classified into the following groups :

1. Data Transfer Group
    # MVI
    # MOV
    # LXI
    # LDA
    # STA
     # LHLD
      # SHLD
     # LDAX
     # STAX
     # XCHG
     # SHLD

2. Arithmatic Group
    # INR r                           # ADC r
     # INR M                          # ACI data
     # INX rp                          # DAD rp
    # DCR r                           # SUB r
    # DCR M                        # SUB M
    # DCX rp                        # SUI data
    # ADD r                           # SBB r
    # ADD M                        # SBB M
    # ADI data                       # SBI data
    # ADC M                        # DAA

3. Logical Group
    # ANA r                           # CMP M
    # ANA M                        # CPI data
    # ANI data                      # STC
    # XRA r                          # CMC
    # XRA M                        # CMA
    # XRI data
    # ORA r
    # ORA M
    # ORI data
    # CMP r

4. Branch Group
    # JMP
    # JC
    # JNC
    # JZ
    # JNZ
    # JP
    # JM
    # JPE
     # JPO

5. Rotate Group:
    # RLC
    # RRC
    # RAL
    # RAR

6. Machine Control:
    # NOP
    # HLT
   
7. Special 16 bit Instructions:
    # PCHL
    # SPHL
    # XTHL

8. Input/Output,Interrupt and stack related Instructions  
    Stack:                          Interupt:                   Input/Output
    # PUSH rp                   # SI                         # IN
     # POP rp                    # DI                        # OUT
     # CALL addr              # RST n
     # RET                         # SIM
                                        # RIM
