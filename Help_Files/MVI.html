MVI :

   This is an abbrevation for move instruction.It is further classified into two sub-categories.
   a) MVI register,data     b) MVI M,data    
       This instruction moves data immediate to register r.In other words it directly moves a single byte of data in the
       register specified.         

   To give an example, consider,  
       MVI A,32H                 ;  This indicates that data 32H is moved directly into register A(accumulator).
   
   Similarly  MVI B,25H
                    MVI E,20H  and so on
   
        MVI M,20H               ;  This instruction moves data 20H into memory location whose address is specified
                                           ;  in HL register pair.
